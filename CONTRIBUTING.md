# Contributing to DeployForge

First off, thank you for considering contributing to DeployForge!

## Development Setup

1. **Fork and clone** the repository.
2. Ensure you have **Node.js 20+** and **npm** installed.
3. Install dependencies: `npm install`.
4. Run infrastructure locally: `docker-compose up -d`.

## Standards

- **Conventional Commits**: Please use conventional commits (e.g., `feat:`, `fix:`, `docs:`).
- **TypeScript**: All new code must be in TypeScript.
- **Linting**: Ensure `npm run lint` passes before submitting a PR.
